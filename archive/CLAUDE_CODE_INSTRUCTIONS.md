# Claude Code 项目指导文件
## Base44 网站 UI 优化项目

> **项目类型:** Base44 无代码平台网站设计优化  
> **用户技能:** 无编程基础  
> **主要目标:** 创建可直接应用到 Base44 的 CSS 代码

---

## 📌 项目背景

### 用户情况
- **平台:** Base44 (无代码建站平台)
- **技能水平:** 完全不懂代码和编程
- **当前问题:** 
  - 每个页面都要单独装修,重复劳动
  - 修改设计需要改很多页面,维护困难
  - 缺乏统一的设计规范

### 核心需求
- 建立统一的设计系统
- 创建可复用的组件库
- 提供可直接复制粘贴的 CSS 代码
- 所有代码必须能在 Base44 中直接使用

---

## 🎯 工作原则

### DO - 必须做的事

1. **生成标准 CSS**
   - 只使用标准 CSS,不使用预处理器(SCSS/LESS)
   - 不使用需要编译的语法
   - 确保代码可直接粘贴到 Base44

2. **保持简单清晰**
   - 使用语义化的 class 名称
   - 添加详细注释
   - 避免复杂嵌套

3. **模块化设计**
   - 每个组件独立完整
   - 组件之间可自由组合
   - 使用 CSS 变量统一管理

4. **提供完整示例**
   - 给出 HTML 使用示例
   - 说明如何在 Base44 中应用
   - 展示效果演示

### DON'T - 绝对不做的事

1. **不使用需要编译的代码**
   - ❌ 不用 React/Vue 组件
   - ❌ 不用 TypeScript
   - ❌ 不用 SCSS/LESS
   - ❌ 不用 npm 包

2. **不使用 Base44 不支持的特性**
   - ❌ 不用自定义 JavaScript (除非极简单的动画)
   - ❌ 不依赖外部框架
   - ❌ 不使用需要服务器端处理的功能

3. **不给出复杂方案**
   - ❌ 不建议用户修改 Base44 底层代码
   - ❌ 不建议复杂的开发流程
   - ❌ 不假设用户有编程能力

## 🎨 设计系统规范

### 当前设计风格

**主题:** 深色主题 + 金色点缀

**配色方案:**
```css
主色:    #FFD700  (金色)
辅助色:  #FFA500  (橙金)
强调色:  #22C55E  (绿色)
背景:    #0A0A0A  (深黑)
卡片背景: #1A1A1A  (次黑)
文字:    #FFFFFF  (白色)
次要文字: #B0B0B0  (灰色)
```

**视觉特点:**
- 现代简约
- 高对比度
- 柔和动画效果
- 专业商务感

### 必须遵循的规范

1. **所有颜色使用 CSS 变量**
   ```css
   /* 正确 */
   color: var(--color-primary);
   
   /* 错误 */
   color: #FFD700;
   ```

2. **所有间距使用标准单位**
   ```css
   /* 正确 */
   padding: var(--space-md);
   
   /* 错误 */
   padding: 17px;
   ```

3. **动画保持流畅自然**
   ```css
   /* 推荐 */
   transition: all 0.3s ease;
   
   /* 避免过快 */
   transition: all 0.1s;
   ```


3. **给出应用步骤**

### 场景2: 用户想创建可复用组件

**用户会说:**
- "我想创建一个按钮样式用在所有页面"
- "这个卡片设计很好,能否做成组件?"

**你应该:**

1. **创建独立组件 CSS**
   ```css
   /* 组件名称 */
   .component-name {
     /* 完整样式 */
   }
   
   /* 组件变体 */
   .component-name.variant {
     /* 变体样式 */
   }
   
   /* 悬停效果 */
   .component-name:hover {
     /* 交互效果 */
   }
   ```

2. **提供使用示例**
   ```html
   <!-- 基本用法 -->
   <div class="component-name">内容</div>
   
   <!-- 变体用法 -->
   <div class="component-name variant">内容</div>
   ```

3. **说明应用方法**
   ```
   在 Base44 中:
   1. 全局 CSS 中添加组件样式
   2. 在需要的地方使用 class
   3. 可在任何页面复用
   ```

### 场景3: 用户想添加动画效果

**用户会说:**
- "能否让卡片有悬停效果?"
- "我想要页面加载时有动画"

**你应该:**

1. **使用纯 CSS 动画**
   ```css
   /* 定义动画 */
   @keyframes fadeIn {
     from { opacity: 0; }
     to { opacity: 1; }
   }
   
   /* 应用动画 */
   .animate-element {
     animation: fadeIn 0.6s ease;
   }
   ```

2. **避免复杂 JavaScript**
   - 优先用 CSS transition
   - 简单的 hover 效果
   - 使用 CSS animation

3. **如果必须用 JS,保持极简**
   ```javascript
   // 仅在必要时使用
   // 代码不超过 10 行
   // 添加详细注释
   ```

## 📚 代码模板库

### 模板1: 基础组件

```css
/* ============================================
   [组件名称]
   ============================================ */

.component-name {
  /* 布局 */
  display: flex;
  
  /* 尺寸 */
  padding: var(--space-md);
  
  /* 颜色 */
  background: var(--bg-secondary);
  color: var(--text-primary);
  
  /* 形状 */
  border-radius: var(--radius-md);
  
  /* 效果 */
  transition: all 0.3s ease;
}

.component-name:hover {
  transform: translateY(-4px);
  box-shadow: var(--shadow-md);
}

/* 响应式 */
@media (max-width: 768px) {
  .component-name {
    /* 移动端调整 */
  }
}
```

### 模板2: 带动画的组件

```css
/* ============================================
   [组件名称] - 带入场动画
   ============================================ */

/* 动画定义 */
@keyframes fadeInUp {
  from {
    opacity: 0;
    transform: translateY(30px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

/* 组件样式 */
.animated-component {
  /* 基础样式 */
  opacity: 0;
  
  /* 应用动画 */
  animation: fadeInUp 0.6s ease forwards;
}

/* 延迟变体 */
.animated-component.delay-1 {
  animation-delay: 0.1s;
}

.animated-component.delay-2 {
  animation-delay: 0.2s;
}
```

### 模板3: 响应式网格

```css
/* ============================================
   响应式网格布局
   ============================================ */

.grid {
  display: grid;
  gap: var(--space-lg);
}

/* 桌面端 - 3列 */
.grid-3 {
  grid-template-columns: repeat(3, 1fr);
}

/* 平板端 - 2列 */
@media (max-width: 1024px) {
  .grid-3 {
    grid-template-columns: repeat(2, 1fr);
  }
}

/* 手机端 - 1列 */
@media (max-width: 768px) {
  .grid-3 {
    grid-template-columns: 1fr;
  }
}
```

## ⚠️ 常见错误和注意事项

### 错误1: 提供了需要编译的代码

❌ **错误示例:**
```scss
// 这是 SCSS,Base44 不支持
$primary: #FFD700;
.button {
  background: $primary;
}
```

✅ **正确示例:**
```css
/* 标准 CSS */
:root {
  --primary: #FFD700;
}
.button {
  background: var(--primary);
}
```

### 错误2: 假设用户懂编程概念

❌ **错误说法:**
"你需要创建一个闭包来处理事件委托"

✅ **正确说法:**
"复制这段代码到页面,当用户点击按钮时就会有效果"

### 错误3: 提供过于复杂的方案

❌ **错误方案:**
"建立 webpack 构建流程,配置 PostCSS,使用 Tailwind..."

✅ **正确方案:**
"把这段 CSS 粘贴到 Base44 的全局样式设置中"
